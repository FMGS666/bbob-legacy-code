COmparing Continuous Optimizers (COCO) Software

provides a set of tools for generating and presenting unified
benchmarking results. COCO provides:

* benchmark functions (detailed description and source code)
* description of the experimental procedure
* tools for post-processing the resulting data into tables and graphs

directories:
  code/c: benchmark functions, example experiment scripts in C and C++
  code/matlab: benchmark functions, example experiment scripts in Matlab/Octave 
  code/java: JNI interfrace to the C-code, example experiment scripts in Java
  code/python: benchmark functions, example experiments scripts in Python
      and post-processing utilities
  templates/latex: templates used for the generation of papers
  doc: documentations
